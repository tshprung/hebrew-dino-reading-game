# אפשרויות ויזואליות לקן דינוזאורים (לבחירת אסט)

המסך משתמש כרגע ב־`DinoNestMark` — ציור קומפוזיט פנימי. כאן רשימת **סגנונות** לבחירת תמונה / איור סופי (להחלפת האסט או כהשראה לאמן).

## 1. קן תלולית (סאורופודים)
- **איך זה נראה:** גבעת עפר או חול עגולה־מעוגלת, ביצים שקועות חלקית בפנים.
- **חיפוש תמונה:** `sauropod nest mound fossil reconstruction`

## 2. קן קערה (מאיאסורה / האדרוזאורים)
- **איך זה נראה:** שקע רדוד בקרקע, ביצים במעגל או ספירלה, לפעמים צמחייה סביב.
- **חיפוש תמונה:** `Maiasaura nest bowl eggs illustration`

## 3. קן שריטה (תרופודים)
- **איך זה נראה:** שריטה / שקע באדמה, מרופד בעלים, ענפים קטנים וביצים במרכז.
- **חיפוש תמונה:** `theropod nest scrape leaves illustration`

## 4. קן “ציפור ענק” (מקלות צפופים)
- **איך זה נראה:** קערה ממקלות דקים מסודרים / מבולגנים, ביצים בפנים — קריא גם בגודל קטן במפה.
- **חיפוש תמונה:** `bird nest illustration stylized game asset`

---

**איך לבחור:** שלחי קישור לתמונה או קובץ PNG/SVG, או כתבי מספר (1–4) + “יותר חום / יותר ירוק” וכו’, ונחבר את האסט ל־`JourneyScreen` / `ChaptersScreen` במקום הציור הקומפוזיט.
